# NinjaSoft FTP 1.0.1

Atualização corretiva da primeira versão pública do NinjaSoft FTP.

## Correção

- o executável agora usa o subsistema gráfico do Windows e não abre uma janela
  de prompt de comando junto com a interface.

Esta versão mantém os recursos da 1.0.0: interface em português do Brasil,
temas automático, claro e escuro e suporte a FTP, FTPS e SFTP.

## Como usar

Baixe `NinjaSoft-FTP-1.0.1-Windows-x64-portable.zip`, extraia todo o conteúdo e
execute `bin\NinjaSoftFTP.exe`.
