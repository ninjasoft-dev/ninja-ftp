# NinjaSoft FTP 1.0.0

Primeira versão pública do cliente FTP, FTPS e SFTP da NinjaSoft para Windows.

## Destaques

- interface em português do Brasil;
- identidade visual NinjaSoft;
- temas automático, claro e escuro;
- navegação local e remota no layout compacto;
- realce consistente dos botões nos temas claro e escuro;
- edição portátil para Windows 64 bits, sem instalador;
- suporte a FTP, FTPS e SFTP herdado da base FileZilla Client.

## Como usar

Baixe `NinjaSoft-FTP-1.0.0-Windows-x64-portable.zip`, extraia todo o conteúdo e
execute `bin\NinjaSoftFTP.exe`. O arquivo `.sha256` permite validar a integridade
do download.

O executável ainda não possui assinatura digital. O Windows pode solicitar
confirmação na primeira execução.
